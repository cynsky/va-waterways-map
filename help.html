Virginia Waterways Trip Planner

Map Application Use

This application is intended for reference use only. There is no assurance of accuracy or completeness of any data displayed in this application.

Adding Overlays - Click the "Overlays" button in the top right corner of the map viewer to access the available map data. Click your selection and that data will be added to the map. Clicking features or locations will return different results depending on whether you click a feature such as a boat ramp or just a location.

  • Esturaries - When this overlay is activated, clicking a bay water body will open a baloon windows showing location and water quality information including a link to a detailed fact sheet of assessed impairments if present.
  • Bay floor contours - When activated, this overlay shows a shaded relief of the floor of esturaires permitting the viewer to ascertain drop offs and points that aren't visible from the water's surface.
  • Hospitals - This overlay displays hospital locations that provide hospital name and link to their website when clicked.
  • Boat Ramps - This overlay dispalys DGIS hosted boat ramp locations that provide additional location information about that boat ramp when clicked.
  • Aerial Imagery - This overlay is the most recent photographic imagery hosted by VGIN.
  • Juristictional Boundaries - This overlay displays locality boundaries such as counties and cities.
  • Marine Navigation Charts - This overlay displays NOAA hosted marine navigation charts for general reference.
  • Road Projects - This overlay displays VDOT hosted data on current road constructions projects in the state of Virginia.
  • Weather Radar - This overlay displays NOAA base reflectivity updated every five (5) minutes.
  • Tide Predictions - This overlay displays NOAA hosted weather stations that provide sunrise/sunset information for that location as well as a link to tide predictions on a daily, weekly, or monthly basis.
  • Click any other location - Clicking anywhere in the map produces a baloon window with the coordinates including a link to detailed weather information such a temperature, barametric pressure and more for that location.

Navigating the map - Users can pan the map by clicking a location and dragging the mouse accross the screen. The "Inset Map" in the bottom left corner of the screen can be used to pan long distances by dragging the red rectangle in your desired direction.
